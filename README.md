# Vue + Hono Typesafe Starter Monorepo w/ RPC, Tanstack query, Tailwind

To install dependencies:

```bash
bun install

cd ui/
bun install
```

This project was created using `bun init` in bun v1.1.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
